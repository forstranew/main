🔵 문제 16
WAR(Web Application aRchive) 파일에 관한 설명으로 맞는 것을 모두 고르시오.
a. JSP 페이지, 서블릿, Java 클래스 파일, HTML 파일 등을 압축한 파일
b. 웹 애플리케이션을 웹 컨테이너에게 배포하기 위해 사용되는 파일
c. 확장자는 jar
d. Eclipse를 사용하면 손쉽게 WAR 파일을 만들 수 있음

① a, b
② a, c, d
③ a, b, d
④ b, c, d
✅ 정답: ③ (a, b, d)
WAR 파일은 웹 애플리케이션을 배포하기 위해 필요한 모든 리소스를 하나로 압축한 파일입니다. - a: JSP, 서블릿, 클래스, HTML, CSS 등 웹 앱의 구성 요소를 포함 - b: 웹 컨테이너(예: Tomcat)에게 배포할 때 사용 - c: WAR 파일 확장자는 .war이며, .jar과는 다름 - d: Eclipse에서 프로젝트 내보내기 기능으로 쉽게 생성 가능
🔵 문제 17
웹 애플리케이션 폴더 구조에서 web.xml 파일이 위치하는 폴더로 빈 부분에 들어갈 이름은 무엇인가?
[톰캣설치폴더]\webapps\[웹애플리케이션이름]\_____\web.xml
① WEB-INF
② JSP
③ META-INF
④ WebContent
✅ 정답: ①
`WEB-INF`는 클라이언트가 직접 접근할 수 없는 보안 영역으로, web.xml(배포 서술자), classes/, lib/ 폴더가 포함되어 있습니다. 클라이언트가 직접 접근할 수 없기 때문에 민감한 설정 파일과 클래스가 안전하게 보호됩니다.
🔵 문제 18
HTTP 요청 방식 중 POST 방식에 관한 설명으로 맞는 것을 모두 고르시오.
a. URL에 쿼리 문자열을 붙여 요청한다.
b. HTTP 메시지의 몸체에 데이터를 전달한다.
c. 많은 양의 데이터를 보낼 때 적당하다.
d. 상대적으로 보안에 취약하다.

① a, c
② a, d
③ b, c
④ b, d
✅ 정답: ③ (b, c)
POST 방식은 HTTP 요청의 본문(body)에 데이터를 담아 서버로 전송합니다. - a: URL에 데이터를 붙이는 것은 GET 방식 - b: POST는 본문 전송 - c: 많은 데이터를 전송 가능 - d: GET보다 URL에 데이터 노출이 없어 상대적으로 안전함
🔵 문제 19
다음의 기능과 관련이 있는 JSP 내장 객체는 무엇인가?
- addCookie(): 쿠키 추가
- setStatus(): 상태 코드 설정
- getWriter(): 출력 스트림 리턴
- sendRedirect(): 다른 페이지로 이동
① out
② request
③ response
④ exception
✅ 정답: ③
제시된 메서드는 모두 response 객체의 메서드입니다. response는 클라이언트에게 보내는 응답을 관리하며, 쿠키 추가, 상태 코드 지정, 출력 스트림 제공, 리다이렉션 등을 수행할 수 있습니다.
🔵 문제 20~22
form.jsp 예제: 
<%@ page contentType="text/html; charset=EUC-KR"%>
<html><body>
<FORM ACTION="MyServlet" METHOD="POST">
    이름: <input type="text" name="name"><br>
    가장 좋아하는 음식:
    <select name="food">
        <option value="피자">피자</option>
        <option value="치킨">치킨</option>
    </select><br>
    <input type="submit" value="확인">
</FORM>
</body></html>
20. ✅ 정답: ② http://localhost:8080/JSP/form.jsp
21. ✅ 정답: ④ MyServlet의 doPost() 메서드 호출
22. ✅ 정답: ② request.getParameter("food")
- Tomcat 기본 포트는 8080, URL 구조는 http://localhost:8080/[웹앱명]/[파일]
- METHOD="POST"이면 doPost() 호출
- HTML 폼에서 name="food"로 지정된 데이터를 읽을 때는 request.getParameter("food") 사용
🔵 문제 23
JSP 페이지의 사용 영역 '속성을 저장하는 영역'과 관계가 있는 JSP 내장 객체가 아닌 것은?
① pageScope
② request
③ session
④ application
✅ 정답: ①
pageScope는 EL(표현 언어)에서 사용되는 객체이며 JSP 내장 객체 pageContext, request, session, application과 달리 JSP 영역 관련 내장 객체가 아닙니다.

</BODY>
</HTML>
27. ✅ 정답: ② <jsp:forward page="to.jsp" />
28. ✅ 정답: ② getAttribute
29. ✅ 정답: ③ 액션 태그
