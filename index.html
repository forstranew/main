문제 36: 다음이 설명하는 HTTP 요청 방식은 무엇인가?
원하는 방식으로 인코딩된 데이터를 요청 메시지의 몸체에 포함하여 전송하면서 자원을 요청하는 경우 사용됨
① GET
② HEAD
③ POST
④ FTP
POST 방식은 데이터를 HTTP 요청 메시지의 본문(body)에 포함하여 전송하는 방식입니다. URL에 데이터가 노출되지 않으며, 대용량이나 민감한 정보 전송 시 사용됩니다.

③ 웹 컨테이너를 서버로 등록시키는 것이다.
④ 버퍼의 내용을 일차적으로 클라이언트에 보내는 것이다.
